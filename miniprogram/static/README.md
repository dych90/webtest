# 静态资源说明

## 需要添加的图片资源

### Logo
- `logo.png` - 应用 Logo（建议尺寸：200x200px）

### Tabbar 图标（建议尺寸：81x81px）
- `tabbar/home.png` - 首页图标（未选中）
- `tabbar/home-active.png` - 首页图标（选中）
- `tabbar/student.png` - 学生图标（未选中）
- `tabbar/student-active.png` - 学生图标（选中）
- `tabbar/course.png` - 课程图标（未选中）
- `tabbar/course-active.png` - 课程图标（选中）
- `tabbar/mine.png` - 我的图标（未选中）
- `tabbar/mine-active.png` - 我的图标（选中）

## 图标设计建议

- 未选中状态：灰色（#999999）
- 选中状态：主题蓝色（#409EFF）
- 格式：PNG（透明背景）
- 尺寸：81x81px（推荐）
